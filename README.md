# tab-tracker

Guitar Tab Tracking web app, built with Vue and Express

## How to?

1. Goto `./server/`
```
npm install
npm start
```

2. Goto `./client/`
```
npm install
npm run dev     // for development mode
npm run build   // create production build in './client/dist/' folder
```

## Stack

- VueJS
- NodeJS with Express
- MySQL with Sequelize
